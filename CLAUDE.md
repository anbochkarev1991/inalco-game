# Project rules — fable-creates-game

## Git workflow (overrides global rules for this project)
- All edits are allowed directly on `main` in this project, at all times.
- This project overrides the global "protected branch" rule: do NOT warn or refuse
  when working on `main` here. Committing and editing on `main` is expected.
- The user still names branches themselves if/when a branch is used — never suggest one.
